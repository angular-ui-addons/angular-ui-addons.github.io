AngularUI Addons

A free set of AngularJS native libraries built with respect to AngularUI project to make your UI look prettier.

Getting started

Dependencies

This repository contains a set of useful AngularJS directives based on Bootstrap's markup and CSS. The library is build with common views in mind, so you can use it without Bootstrap markup, suppressing it with you own templates. The library has no dependency on jQuery or Bootstrap's the JavaScript only is required. Bootstrap library is only used for "e.g." purposes as a markup for initial templates for directives and can be surely replaced with your own. The only required dependencies are:

  • AngularJS (minimal version 1.0.8)

Files to download

The only option for now is to get source code from here.

Whichever method you choose the good news that the overall size of a download is very small: <20kB for all directives (~5kB with gzip compression!)

Installation

As soon as you've got all the files downloaded and included in your page you just need to declare a dependency on the angular-ui-addons.inclist (or which ever you want) module:

angular.module('myModule', ['angular-ui-addons.inclist', ...]);

You can fork one of the plunkers from this page to see a working example of what is described here.

Demo

The widget allow you to place two items: input - the inclist-input directive and list which is inclist-out directive where you need it and implement simple but useful "incremental list" ux-pattern.

Be sure to set inclist-input-list and inclist-input-prop-selection attributes for inclist-input directive to those models that should be responsible for list content and current input. For inclist-out directive set attribute inclist-out-list that should be assigned to the same model as inclist-input-list for list output to work properly.
